Standard Blocks Reference

filter

Parameters

filter-name
The name of the filter that you want to apply

Description

Applies any filter that is defined on the current context to all the contents of the rendered block. 
{% filter escape %}
<h1>Some title</h1>
<p>Since this is passed through the 
"escape" filter the < and > characters 
will be converted to &gt; and &lt; 
as you would expect them to if 
passed through an inject tag with 
the "escape" filter.</p>
{% end %}

&lt;h1&gt;Some title&lt;/h1&gt;
&lt;p&gt;Since this is passed through the 
&quot;escape&quot; filter the &lt; and &gt; characters 
will be converted to &amp;gt; and &amp;lt; 
as you would expect them to if 
passed through an inject tag with 
the &quot;escape&quot; filter.&lt;/p&gt;

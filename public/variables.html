Functional Variables Reference

load

Parameters

filename
The path to the file you want to load

Description

Loads the given filename and injects it into the rendered template without performing any parsing or rendering. 
{{ load "partials/load-example.cadenza" }}

{% for letter in alphabet %}{{ letter }}{% endfor %}

render

Parameters

filename
The path to the file you want to load

Description

Loads the given filename and renders it using the same context is it was called with. 
{{ render "partials/load-example.cadenza" }}
